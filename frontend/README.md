# 🔥 Sentient Shield Vault

> **Self-Operating DeFi Engine powered by AI, Security & Reputation**

A next-gen DeFi vault that eliminates centralized automation by combining **on-chain execution, AI-driven risk control, and identity-based trust systems**.

---

## 🚀 Features

* ⚡ **Fully On-Chain Operations**
  No off-chain bots. Deposit, harvest, rebalance — everything happens on-chain.

* 🛡️ **ShieldBot Security Layer**
  AI-based risk scoring system that blocks high-risk transactions.

* 🔐 **NexaID Verification**
  On-chain identity + reputation score for enhanced trust and rewards.

* 🎮 **Gamification System**
  Earn XP, level up, and unlock yield bonuses.

* 💰 **Dynamic Yield Engine**
  Automated yield generation and reward distribution.

* 🏆 **Bounty System**
  Users earn rewards for maintaining vault efficiency.

---

## 🧠 Tech Stack

* **Frontend:** React + Ethers.js
* **Smart Contracts:** Solidity (Hardhat)
* **Blockchain:** HashKey Testnet
* **Security Layer:** ShieldBot (AI Simulation)
* **Identity Layer:** NexaID

## 🔧 How It Works

1. Connect wallet
2. Verify via NexaID
3. Deposit USDC into vault
4. Earn yield + rewards
5. Harvest, claim, or rebalance
